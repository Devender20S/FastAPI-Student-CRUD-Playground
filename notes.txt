SGI: server gateway interface : protocols of communication between API code and server. 2 types -->ASGI(used in fastapi: starlette) and WSGI (used in flask: werkzeug)
    WSGI: WEB server gateway interface= Folows sync nature matlab ek baar mein ek hi request process krr skte ho. iska blocking nature hota hai . slow hota hai  
    ASGI:async server gateway interface= yeh web server aur api code ke beech mein 2 way communication establish krta hai. koi message agar http request ke form mein jaa rhi hai toh woh usse JSON formatt mein convert krta hai python ki understanding ke liye and vice versa.
