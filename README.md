# Student Management System - FastAPI + React CRUD

A simple full-stack CRUD application to learn and demonstrate core features of **FastAPI**.

### Tech Stack
- **Backend**: FastAPI + Python
- **Database**: MySQL
- **Data Validation**: Pydantic
- **API Documentation**: Swagger UI (automatic)
- **Frontend**: React (basic implementation)

### Main Learning Objectives Achieved
- Creating RESTful APIs with FastAPI
- Using Pydantic models for request/response validation
- Database operations using SQLAlchemy + MySQL
- Automatic interactive API documentation (Swagger UI)
- Basic React frontend with CRUD operations
- Connecting React frontend with FastAPI backend

### Features
- Create, Read, Update and Delete student records
- Student fields: name, roll number, email, age, grade, etc.
- Clean and modern Swagger UI documentation
- Simple and clean React interface
